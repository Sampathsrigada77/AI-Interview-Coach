## AI Interview Coach
A full-stack AI-powered interview preparation platform.

## Features
- Generate interview questions
- AI-based answer evaluation
- Score and feedback system

## Tech Stack
- React
- Node.js
- FastAPI
- Gemini API

## Architecture
React → Node → FastAPI → Gemini

## How to Run
### 1. Start FastAPI
cd ai-service
uvicorn main:app --reload

### 2. Start Node
cd backend
node server.js

### 3. Start React
cd frontend
npm start
